Currently, I have made Codeforces-scrapper with some features like Code submitter, Profile access, Profile Comparator, Upcoming and Past Contest Information.

## How to Start Project

Fork the repository and download it through zip.

## Installation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements.txt

```bash
pip install -r requirements.txt
```

## Chrome Driver
Download chrome driver for codeforces code submitter as it will work then only, if you are using firefox then download gecko driver(Link I have provided below), but I suggest chrome driver for online submitter.

If using chrome of 125. version then I have provided chrome driver for it in repository.


```
Chrome Driver : https://googlechromelabs.github.io/chrome-for-testing/
```
```
Gecko Driver : https://firefox-source-docs.mozilla.org/testing/geckodriver/index.html

```

## Usage

For Code submission through automation, use read code method in codeforces_code_submmiter.py and give your code.

In codeforces_scrapper.py give the handle of your codeforces Id and take the required output by the methods
